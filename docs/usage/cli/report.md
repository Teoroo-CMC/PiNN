# log

Inspect the status and summary of models.

## Usage

```bash
pinn report models_dir [options]
```

## options
| Option [shorthand] | Default    | Description               |
|--------------------|------------|---------------------------|
| `--fliter [-f]`    | ``         | only print keywords in model path |
| `--l [-l]`         | `'eval.log'` | log file name |
